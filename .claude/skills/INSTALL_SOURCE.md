# Источник скиллов

Установлено из плагина `ui-ux-pro-max` маркетплейса
`nextlevelbuilder/ui-ux-pro-max-skill` (v2.11.0, MIT).
Репозиторий: https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

Плагин объявляет `"skills": "./.claude/skills/"`, поэтому установлены все
входящие в него скиллы. Обновление — заново скопировать `.claude/skills/`
из репозитория плагина.
